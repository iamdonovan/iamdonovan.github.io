setting up github

The files for this module (practical workbooks, example scripts, and setup files) will all be shared via GitHub, which means you will need to create an account if you do not already have one. You can do so by clicking ‘Sign up’ in the upper right corner of the GitHub homepage (https://github.com):

Make sure that you do this before you move on to the next step, installing git.
